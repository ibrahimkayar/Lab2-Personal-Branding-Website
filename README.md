# Personal Branding Website with Vibe Coding

## Overview

This project was developed for **Samsung Innovation Campus – Generative AI Lab 2**.

The objective of this project is to create a personal branding website using Generative AI through an end-to-end workflow.

---

## Workflow

- Personal Profile Definition
- AI-generated Images
- Website Generation using HTML & CSS
- Website Refinement

---

## Technologies

- HTML5
- CSS3
- ChatGPT
- Gemini AI

## AI Prompts

### Text Generation

Organize the following information into three sections:
- Who I Am
- My Skills
- My Future Goal

Write it in a professional and student-friendly tone.

---

### Image Generation

Generate professional illustrations representing:

- Who I Am
- My Skills
- My Future Goal

---

### Website Generation

Create a modern and responsive personal portfolio website using HTML and CSS.

Requirements:

- Responsive
- Blue & White Theme
- Cards Layout
- Professional Design
- Data Science Portfolio

---

## Result

The final website presents my personal profile, technical skills, career goals, and AI-generated illustrations in a modern responsive portfolio.
